How to Use This Tool

  1. Navigate views using the tabs at the top.
  2. Create items via "＋ New item" or "＋ Add item" in any workstream column.
  3. Edit items by clicking "✎ Edit item" on expanded cards, or clicking rows in Swimlane/People views.
  4. Quick-update status using status buttons on expanded cards.
  5. Add notes via "✎ Add note" on expanded cards.
  6. Export via "↗ Export" for Slack, Email, Weekly, or CSV.

Tabs at a glance

  • Home — Role-aware status (Exec / Operator toggle) with "since last visit" ribbon.
  • Focus — Items sorted by attention score (blockers, overdue, stale, just-unblocked, decisions).
  • Work — Board + Swimlane views (toggle at top).
  • People — Owner rollup with workload, personal next-step queues, stale-owner flags.
  • Dependencies — Default view for everyone; Workbench for sequencers.
  • Weekly Report — Exec narrative + W-o-W deltas.

Status Meanings

  • Todo (gray) — Not started.
  • In Progress (amber) — Active work.
  • Blocked (red) — Waiting on dependency.
  • Done (green) — Complete.

Types: Action / Blocker / Gate / Sequenced

  • Action — Work that can proceed now.
  • Blocker — Blocks other work.
  • Gate — Launch gate / approval checkpoint.
  • Sequenced — Awaiting prerequisites.

Attention Score

Every non-done item is scored automatically from: blocker status, due-date pressure, staleness, just-unblocked status, gate-gating, and decision flags. Critical = 🔥, Watch = ⚠, Normal = ·. Used for Focus queue ordering and Home priority.

Notes — @mentions and decisions

Typing @Name in notes adds that item to their Focus queue. Phrases like "decision needed", "waiting on", "sign-off", or "go/no-go" are auto-flagged as decisions. Or tick Decision required explicitly on the edit form.
